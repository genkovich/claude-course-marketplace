# pr-review-rules

## Purpose

Make PR review consistent across the team. Same five rules every reviewer applies, so authors stop guessing what "looks good to me" means and reviewers stop arguing about style.

## Install

```
/plugin install pr-review-rules@team-marketplace
```

## Skills

- `review-checker` — auto-triggers on PR review intents. Enforces five rules: changelog entry, no debug calls, test coverage on new branches, no unrelated formatting churn, security checklist for auth/PII paths.
