# claude-course-marketplace

Internal Claude Code plugin marketplace for the Engineering Platform team at
Acme. Bundles opinionated workflow plugins — pre-deploy checklists, PR review
rules — so the team installs one marketplace and gets the whole house style.

Releases follow [Semantic Versioning](https://semver.org/) and ship with a
changelog entry under [CHANGELOG.md](./CHANGELOG.md). Tone for release notes:
concise, user-facing bullets that a non-engineering reviewer can skim — no
internal Jira IDs, no commit hashes, no marketing fluff.
